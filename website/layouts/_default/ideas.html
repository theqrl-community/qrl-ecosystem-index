How to choose

Coverage Checklist

  • Does it make QRL easier to use, build on, monitor, or explain?
  • Does it fill a category or project type with few or no current listings?
  • Can it be maintained by a small team or community contributor?
  • Does it have a clear first version that can ship on testnet?
  • Can the project publish enough information for others to evaluate and list it?
