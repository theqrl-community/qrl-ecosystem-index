MyQRLWallet is a DigitalGuards web wallet for QRL 2.0. It provides a
browser-based interface for creating accounts and managing transactions on
the quantum-resistant ledger.

The wallet is part of the DigitalGuards QRL Zond ecosystem and links to a
public frontend repository.
