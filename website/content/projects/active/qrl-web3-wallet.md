QRL Web3 Wallet is an open-source browser extension wallet developed by The
QRL for QRL 2.0. It lets users create and import accounts, send QRL
transactions, connect to dApps through EIP-1193 and EIP-6963 provider
support, and manage QRC-20 tokens.

The project README notes that the code relates to the QRL v2.0 testnet and
is not suitable for the current QRL v1 mainnet. Pre-built extension bundles
are published through the project's GitHub releases.
